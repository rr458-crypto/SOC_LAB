How to create a Node.js web server using Express

Step 1: Create project folder
mkdir myWebServer
cd myWebServer
npm init -y 
npm i express@4.21.2   

Step 2: Create folder for static files
>mkdir public
Put any HTML, CSS, JS files you want to serve in the public folder.

Step 3: Create server.js

const express = require('express');
const path = require('path');
const app = express();
const port = 5000;  // Your server port

// Serve static files from "public" folder
app.use(express.static(path.join(__dirname, 'public')));

// Optional: default route
app.get('/', (req, res) => {
  res.sendFile(path.join(__dirname, 'public', 'index.html'));
});

// Start the server
app.listen(port, () => {
  console.log(`Server running at http://127.0.0.1:${port}`);
});

Step 4: Test the server
Run the server:
node server.js

Open a browser and go to:
http://127.0.0.1:5000
